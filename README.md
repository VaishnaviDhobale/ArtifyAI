# ArtifyAI
ArtifyAI is a website where users can enter a text prompt, and based on that input, it generates a unique image. Users can also download the generated images and use them whenever they want.

## Tech Stack 
1. JavaScript
2. React
3. Tailwind CSS
4. Express
5. MongoDB
6. Git & GitHub
7. Vercel
